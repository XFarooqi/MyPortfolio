HTML Cheatsheet

Basic HTML Codes


  1. Heading Tags: This Tag is used for Headings. 
    
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

    Output:


    Heading 1

    Heading 2

    Heading 3

    Heading 4

    Heading 5
    Heading 6


  2. Paragraph Element: This will be used to write a parahpargh using 
    
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

    Output:

    This is a paragraph.I can write whatever I want!!!

    This is another paragraph.

  3. Image Tag This tag shows how to insert an image.

  6. Span Tag: Span is an inline element where all the content will be written in a single line. 
    
<span style="background-color: steelblue; border-radius: 2;">This is Span 1</span>
<span style="background-color:blueviolet;">This is Span 2</span>

    Output:

    This is Span 1 This is Span 2


  7. Div Tag Div is a block element and it takes a complete block of space for itself. It means anything after the Div will come in the next block. 
    
<div style="background-color: red; border: 1px solid black;"> Div1</div> 
<div style="background-color: green; border: 1px solid black;"> Div2</div>

    Output:

    Div1
    Div2
